电脑如何翻墙？2026年Windows与Mac科学上网新手指南

在电脑（PC/Mac）上“翻墙”或“科学上网”早已成为很多人的日常需求，无论是为了查阅技术文档、使用 Google 服务，还是观看 YouTube、Netflix 等流媒体内容。

如果你是第一次尝试在电脑上配置翻墙环境，可能会被各种专业术语（如 Clash、节点、订阅、代理）搞得晕头转向。别担心，本文将用最通俗易懂的语言，手把手教你在 Windows 和 Mac 电脑上实现科学上网。

核心概念：软件与节点

在开始之前，你只需要理解两个核心概念，就能明白翻墙的原理：

  1. 客户端（软件）：这是一辆“车”。
    • 你需要下载并安装在电脑上的软件，比如 Clash Verge Rev。
    • 软件本身是免费的，但光有软件（车）没有油是跑不起来的。
  2. 订阅/节点（机场）：这是“油”。
    • 你需要购买的服务，通常称为“机场托管”或“订阅链接”。
    • 你会得到一个以 http 开头的链接，把它导入到软件里，你的“车”才能跑起来，带你访问外网。

还没有节点（油）？

如果你还没有购买节点服务，可以参考我们整理的 便宜好用的翻墙机场推荐评测，选择一个稳定且适合你的服务商。

一、 Windows 电脑如何翻墙

Windows 是目前使用最广泛的系统，翻墙方式也非常成熟。目前最推荐使用的是开源、免费且功能强大的 Clash Verge Rev。

1. 下载与安装

不要去下载老旧的 Clash for Windows（作者已停更），推荐使用新版的 Clash Verge Rev。

  • 详细教程与下载地址：请查看 Windows/Linux/Mac 安装 Clash Verge（含订阅配置教程）

2. 简易操作步骤

  1. 下载软件：从上述链接下载 .exe 安装包并安装。
  2. 购买订阅：从机场服务商处复制“Clash 订阅链接”。
  3. 导入配置：打开 Clash Verge Rev，点击左侧的 订阅 (Subscription)，粘贴链接并点击 导入 (Import)。
  4. 开启代理：
    • 点击左侧 代理 (Proxies)，选择 Rule (规则模式，推荐) 或 Global (全局模式)。
    • 点击 设置 (Settings)，打开 System Proxy (系统代理) 开关。
    • 重要：看到系统代理开关变绿，才算真正翻墙成功。

二、 Mac (macOS) 电脑如何翻墙

苹果电脑（MacBook/iMac）的安全性较高，安装软件时可能遇到“无法打开”的提示，但配置过程与 Windows 类似。

1. 下载客户端

Mac 用户同样推荐使用 Clash Verge Rev（支持 Intel 和 M 芯片）或经典的 ClashX Pro。

  • 详细教程与下载地址：请查看 macOS ClashVergeRev 安装与使用指南

2. 简易操作步骤

  1. 安装软件：下载 .dmg 文件，将图标拖入应用程序文件夹。如果提示“损坏”或“无法验证开发者”，请到 系统设置 -> 隐私与安全性 中点击 仍要打开。
  2. 导入订阅：
    • 复制机场的订阅链接。
    • 打开软件，找到 配置 或 订阅 栏目，粘贴链接并下载配置。
  3. 启动连接：
    • 在状态栏（屏幕右上角）找到软件图标。
    • 勾选 设置为系统代理 (Set as System Proxy)。
    • 建议开启 增强模式 (Enhanced Mode) 以获得更好的体验。

三、 常见问题 (FAQ)

Q1: 软件打开了，为什么还是上不了网？

  • 检查系统代理：确保软件内的“系统代理 (System Proxy)”开关已开启。
  • 检查模式：一般选择“规则模式 (Rule)”。如果想让所有流量都走代理，可以临时切到“全局模式 (Global)”测试。
  • 检查节点：在代理界面点击测速（通常是闪电图标），如果节点全是超时 (Timeout)，说明你的订阅过期了或机场跑路了。

Q2: 关闭软件后，电脑无法上网了怎么办？

  • 这是新手最容易遇到的问题。Clash 必须先关闭“系统代理”开关，再退出软件。
  • 解决方法：
    1. 重新打开 Clash 软件。
    2. 关闭“系统代理”开关。
    3. 正常退出软件。
    • 或者：进入 Windows 设置 -> 网络和 Internet -> 代理，手动关闭“使用代理服务器”。

Q3: 手机和电脑可以共用一个订阅吗？

  • 可以。大部分机场支持多设备同时在线（通常限制 3-5 台设备）。你只需要把同一个订阅链接分别导入手机和电脑的客户端即可。
  • 手机翻墙教程请参考：手机如何翻墙？Android与iOS教程

总结

电脑翻墙其实就三步：下载软件 -> 导入订阅 -> 开启系统代理。
