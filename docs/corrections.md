# 更正与反馈政策

> 发现价格变化、链接失效、版本过期或事实错误？欢迎指出。清晰、可复查的反馈能让修正更快完成。

::: card-grid cols="3"

- **1. 页面地址**

  附上出现问题的完整页面链接。

- **2. 具体问题**

  指出原文位置，以及你认为不准确的内容。

- **3. 可复查来源**

  提供官方页面、公告、截图日期或其他原始材料。

:::

## 收到反馈后如何处理

本站会区分事实更正、观点分歧和无法验证的主张。确认影响结论的错误后，会修改正文并更新修改日期；重大变化会在文章中保留简短说明。仅调整错别字或排版时，不应将页面标记成经过实质更新。

服务商可以提交事实材料，但不能要求删除真实的负面记录或以商业合作换取结论修改。涉及用户隐私、凭证或订阅信息时，请先完成脱敏。

::: tip 隐私提醒
提交截图前，请遮挡邮箱、订单号、订阅地址、访问令牌、付款信息和其他个人资料。
:::
