# 商业合作与推广链接披露

::: info 一句话说明
本站部分机场、软件或服务链接含推广参数。你通过链接注册或购买后，本站可能获得佣金，通常不会增加你支付的价格。
:::

::: card-grid cols="3"

- **💰 可能获得佣金**

  推广关系会公开说明，不把商业链接伪装成普通引用。

- **⚖️ 不购买排名**

  佣金不应改变事实、风险提示和负面状态更新。

- **🛡️ 不提供担保**

  第三方价格、退款、隐私和存续风险仍由服务方承担。

:::

## 推广关系不会改变什么

推广关系不代表本站为第三方服务提供担保，也不应影响事实描述、风险提示和负面信息更新。服务出现重大故障、停止运营或其他高风险信号时，本站会优先提醒读者，并视情况停止推荐。

本站不会因为商家付费而把未经验证的宣传写成实测结论。免费试用、测试账号、赞助或其他可能影响判断的关系，原则上也应在相关页面说明。

::: warning 购买前检查
请核对最新套餐、退款政策和服务条款，并优先选择短周期方案。历史表现和本站推荐都不能消除第三方服务风险。
:::
