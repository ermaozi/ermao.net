### 方法一：电脑（Windows / Mac）翻墙看 YouTube

电脑上推荐使用免费开源的 **Clash Verge Rev** 客户端，操作步骤如下：

#### 步骤 1：购买机场节点

前往 [2026年便宜好用的翻墙机场推荐](/posts/vpn/)，选择一个适合你的机场，注册并购买套餐后，复制你的 **订阅链接**（一般在机场官网的"我的订阅"页面可以找到）。

#### 步骤 2：下载并安装 Clash Verge Rev

参考详细教程👉 [Windows/Linux/Mac 安装 Clash Verge（含订阅配置教程）](/article/0gematwc/)

**快速下载：**
- Windows 用户：[点击下载 Clash Verge Windows x64](https://file.ermao.net/files/clash-verge-rev/Clash.Verge.Windows.x64.exe)
- Mac 用户（Intel）：[点击下载](https://file.ermao.net/files/clash-verge-rev/Clash.Verge.Mac.x64.dmg)
- Mac 用户（M系列芯片）：[点击下载](https://file.ermao.net/files/clash-verge-rev/Clash.Verge.Mac.aarch64.dmg)

#### 步骤 3：导入订阅并开启代理

1. 打开 Clash Verge Rev
2. 点击左侧菜单的「**订阅**」
3. 将你复制的订阅链接**粘贴**进去，点击「**导入**」
4. 点击左侧「**代理**」，选择 **Rule（规则模式）**
5. 点击「**设置**」，打开 **System Proxy（系统代理）** 开关
6. 开关变绿 = 翻墙成功 ✅

#### 步骤 4：打开 YouTube 观看视频

打开浏览器，访问 [https://www.youtube.com](https://www.youtube.com)，搜索「极客湾」即可找到被下架的视频。
