## AI人工智能工具推荐：生产力革命 {#ai-tools}

:::

## 社交与社区类：连接全球视野 {#social-media}

跳出信息茧房，看看世界各地的人们在讨论什么。

### 1. Telegram (电报)
![Telegram 官网首页 logo](https://image.ermao.net/images/blog/gfwwebstie/image-3.png)
- **官网地址：** [https://telegram.org](https://telegram.org)
- **核心定位：** 强调隐私与速度的即时通讯软件。
- **必看理由：** 拥有极其丰富的“频道”和“群组”生态，涵盖技术、新闻、资源分享等领域。支持发送超大文件，且聊天记录云端永久保存。
::: tip 使用技巧
 善用 Bot（机器人）功能，如 RSS 订阅机器人、文件转换机器人；注意隐私设置，隐藏手机号码。
:::

### 2. X (原 Twitter)
![X (Twitter) 社交平台 logo](https://image.ermao.net/images/blog/gfwwebstie/image-1.png)
- **官网地址：** [https://x.com](https://x.com)
- **核心定位：** 全球新闻热点与舆论的爆发地。
- **必看理由：** 信息传播速度极快。关注科技大佬、政治人物或官方账号，能第一时间获取一手资讯。
::: tip 使用技巧
 善用 `List` (列表) 功能对关注者进行分类（如“科技新闻”、“设计灵感”），避免时间线杂乱；搜索时使用 `#hashtag` 追踪话题。
:::

### 3. Reddit
![Reddit 社区首页 logo](https://image.ermao.net/images/blog/gfwwebstie/image-7.png)
- **官网地址：** [https://www.reddit.com](https://www.reddit.com)
- **核心定位：** 互联网的“头版”，基于兴趣的小组讨论社区。
- **必看理由：** 类似于“贴吧”但质量更高。拥有数万个活跃的 Subreddit（子版块），无论你喜欢冷门游戏、机械键盘还是深度学习，都能找到组织。
::: tip 使用技巧
 推荐订阅：`r/technology` (科技), `r/worldnews` (世界新闻), `r/AskReddit` (问答)。按 `Top - All Time` 浏览可以快速看到该版块最精华的内容。
:::

### 4. Instagram
![Instagram 社交应用 logo](https://image.ermao.net/images/blog/gfwwebstie/image-8.png)
- **官网地址：** [https://www.instagram.com](https://www.instagram.com)
- **核心定位：** 视觉导向的社交分享平台。
- **必看理由：** 摄影师、设计师、时尚博主的聚集地。界面极简，内容以高质量图片和短视频（Reels）为主，是寻找审美灵感的好去处。
::: tip 使用技巧
 关注标签（Hashtag）而不仅仅是用户；利用“收藏”功能建立自己的灵感库。
:::

### 5. Discord
![Discord 聊天软件 logo](https://image.ermao.net/images/blog/gfwwebstie/image-9.png)
- **官网地址：** [https://discord.com](https://discord.com)
- **核心定位：** 游戏玩家与兴趣社群的实时语音/文字聊天工具。
- **必看理由：** 很多 Web3 项目、开源软件、游戏公会都将 Discord 作为官方社区。支持高质量语音通话和屏幕分享。
::: tip 使用技巧
 加入 Midjourney 等 AI 绘画服务器体验最新技术；配置服务器通知设置，避免消息轰炸。
:::

体验高质量的流媒体服务与创意内容。

- **必看理由：** 无论是 4K 纪录片、硬核大学课程（如 MIT 公开课）、还是娱乐整活，这里应有尽有。推荐算法非常精准，能帮你发现新兴趣。

## 开发与学术类：提升硬核技能 {#development}

这一类网站是程序员、科研人员和学生群体的宝库，能极大拓展你的知识边界。

### 1. GitHub
![GitHub 代码托管平台 logo](https://image.ermao.net/images/blog/gfwwebstie/image.png)
- **官网地址：** [https://github.com](https://github.com)
- **核心定位：** 全球最大的开源代码托管与协作平台。
- **必看理由：** 它是软件开发的“圣地”。你可以在这里找到几乎所有知名开源项目的源码（如 Linux, React, TensorFlow）。不仅是代码仓库，更是展示个人技术能力的最佳简历。
::: tip 使用技巧
 善用搜索语法（如 `stars:>1000 language:python`）快速筛选优质项目；关注 `Trending` 榜单了解技术风向；利用 GitHub Actions 自动化你的工作流。
:::

### 2. Google Scholar (谷歌学术)
![Google Scholar 学术搜索 logo](https://image.ermao.net/images/blog/gfwwebstie/image-6.png)
- **官网地址：** [https://scholar.google.com](https://scholar.google.com)
- **核心定位：** 免费且强大的学术文献检索引擎。
- **必看理由：** 相比国内搜索引擎，它能提供更纯净、更全面的英文及中文学术资源，支持一键生成引用格式（BibTeX, EndNote 等），是写论文、做研究的必备工具。
::: tip 使用技巧
 创建“个人学术档案”来追踪你的论文引用情况；使用“创建快讯”功能，当有特定关键词的新论文发表时，第一时间推送到你的邮箱。
:::

### 3. Google (谷歌搜索)
![Google 搜索引擎首页 logo](https://image.ermao.net/images/blog/gfwwebstie/image-2.png)
- **官网地址：** [https://www.google.com](https://www.google.com)
- **核心定位：** 全球最强大的综合搜索引擎。
- **必看理由：** 搜索结果精准、广告相对较少且标记明显。其生态圈包含 Gmail、Google Drive、Google Maps 等顶级服务，是获取信息最直接的入口。
::: tip 使用技巧
 掌握高级搜索指令：`site:zhihu.com` (站内搜索), `filetype:pdf` (搜索特定文件), `related:example.com` (查找类似网站)。
:::

### 4. Stack Overflow
![Stack Overflow 问答社区 logo](https://image.ermao.net/images/blog/gfwwebstie/image-5.png)
- **官网地址：** [https://stackoverflow.com](https://stackoverflow.com)
- **核心定位：** 全球最大的程序员问答社区。
- **必看理由：** 遇到代码报错？99% 的概率这里已经有了高质量的解答。它不仅解决问题，还能让你学习到更优雅的编程规范。
::: tip 使用技巧
 提问前先搜索；关注高分回答（Accepted Answer）的同时，也要看评论区，往往有更新的解决方案。
:::

### 5. Dropbox
![Dropbox 云存储服务 logo](https://image.ermao.net/images/blog/gfwwebstie/image-4.png)
- **官网地址：** [https://www.dropbox.com](https://www.dropbox.com)
- **核心定位：** 体验极佳的云存储与同步服务。
- **必看理由：** 以“无感同步”著称，文件版本控制功能强大。适合跨设备办公、备份重要资料。
::: tip 使用技巧
 利用“局域网同步”加速同一网络下的设备传输；结合 IFTTT 实现自动化文件归档。
:::

## 如何高效使用这些网站（进阶技巧） {#tips}

1.  **善用浏览器翻译插件**  
    虽然很多网站支持中文，但英文内容往往更丰富。推荐安装 **沉浸式翻译** 或 **Google Translate** 插件，实现双语对照阅读，既获取信息又学习英语。

2.  **账号安全与隐私保护**  
    - 注册国外服务推荐使用 **Gmail** 或 **Outlook**，避免使用国内邮箱导致收不到验证码。
    - 开启 **2FA (双重验证)**，推荐使用 Google Authenticator 或 Authy，比短信验证更安全。

3.  **网络环境优化**  
    - 访问视频类网站（YouTube, Netflix）时，选择**低延迟、大带宽**的节点。
    - 访问文字类或搜索类网站（Google, Wikipedia），普通节点即可。
    - 使用 **Global (全局)** 模式可能导致国内网站访问变慢，日常建议使用 **Rule (规则/PAC)** 模式。

## 常见问题 (FAQ) {#faq}

**Q：为什么我翻墙后还是打不开 Google 或 YouTube？**  
A：可能是你的代理工具未正确配置，或者节点已失效。请检查：
1. 代理软件是否开启了“系统代理”。
2. 浏览器是否安装了冲突的插件。
3. 尝试更换其他节点或线路。

相关阅读: [便宜好用的翻墙机场推荐评测(长期更新 欢迎推荐)](https://www.ermao.net/posts/vpn/)

**Q：注册这些网站需要国外手机号吗？**  
A：大部分不需要。Google、Twitter 等有时会要求验证手机号，部分国内手机号（+86）可用，但建议使用 Google Voice 或接码平台以保护隐私。

相关阅读: [9esim eSIM 虚拟手机号使用指南](https://www.ermao.net/article/9esim/)

**Q：翻墙浏览这些网站会被追踪吗？**  
A：正规的 HTTPS 网站（浏览器地址栏有小锁）会加密你的传输内容，运营商无法看到你在看什么具体页面，只能看到你连接了代理服务器。但请务必遵守当地法律法规，**不信谣、不传谣、不发布敏感言论**。
