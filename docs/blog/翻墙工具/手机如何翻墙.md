## 翻墙前的准备工作

在开始操作手机之前，你需要明白两个核心概念：**客户端** 和 **节点（订阅）**。

1.  **客户端（软件 / APP）**：就像是一个浏览器，你需要安装在手机上。
    *   **Android (安卓)**：通常使用 **Clash for Android** 或 **Clash Meta**。
    *   **iOS (苹果 iPhone/iPad)**：最常用的是 **Shadowrocket (小火箭)**。
2.  **节点（机场 / 服务器）**：你需要购买或获取一个“订阅链接”，导入到客户端中才能连上网络。这就好比有了手机（客户端），还需要插卡（节点）才能打电话。

::: tip 还没有节点？
如果你还没有好用的节点，可以参考我们的 [便宜好用的翻墙机场推荐评测](/posts/vpn/)，选择一个适合你的服务商。
:::

## 一、 Android (安卓) 手机如何翻墙

安卓手机翻墙相对简单，因为可以直接安装 APK 文件，不需要像苹果那样通过应用商店折腾账号。

### 1. 下载客户端
我们推荐使用 **Clash Meta For Android**，它是目前功能最强大且更新及时的客户端。

*   **详细教程**：[Android手机使用clash指南](/article/eh8f4n86/)

### 2. 简易步骤
1.  **下载安装包**：从上述详细教程中获取 APK 链接并安装。
2.  **导入订阅**：打开软件，点击“配置” -> “从 URL 导入”，将你购买的机场订阅链接粘贴进去。
3.  **启动连接**：回到主界面，点击“启动”按钮（通常是一个圆圈或开关），首次连接会弹出 VPN 授权请求，点击“确定”即可。

苹果手机由于原本的 App Store（中国区）下架了相关软件，所以需要使用 **非中国区（如美区）的 Apple ID** 才能下载。

### 1. 准备美区 Apple ID
你不需要重新注册，可以使用我们提供的免费共享账号直接登录 App Store 下载。

*   **获取账号**：[2026最新免费美区Apple ID共享账号](/blog/freeappleid/)

::: danger 注意事项
**千万不要在系统的“设置”里登录共享账号！** 
只能在 **App Store** 里面登录，否则可能导致手机被锁！

### 2. 下载 Shadowrocket (小火箭)
登录美区账号后，搜索 **"Shadowrocket"** 并下载（注意图标是一个发射的小火箭，不要下错）。

*   **详细教程**：[Shadowrocket(小火箭)新手使用教程：下载安装与配置](/article/z747kgjd/)

### 3. 简易步骤
1.  **切换账号**：在 App Store 退出自己的账号，登录共享美区 ID。
2.  **安装软件**：搜索下载 Shadowrocket，下载完后记得切回自己的账号。
3.  **配置节点**：
    *   方法A（推荐）：在机场后台点击“一键导入 Shadowrocket”。
    *   方法B：复制订阅链接，打开小火箭，它会自动识别并提示添加，或者点击右上角“+”号手动添加。
4.  **开启开关**：点击首页的连接开关，允许创建 VPN配置，输入锁屏密码即可。

## 常见问题解答 (FAQ)

**Q: 翻墙后网速慢怎么办？**
A: 请检查你的节点状态，尝试在客户端内进行“测速”并切换到低延迟的节点。同时，由于网络波动，晚高峰时期可能会有所拥堵。

**Q: 只有 WiFi 能翻，流量不行？**
A: 请检查手机设置中是否允许了该 APP 使用移动数据，或者检查客户端的“代理模式”设置。

**Q: 免费的 VPN 能用吗？**
A: 市面上有许多宣称免费的 VPN，但它们往往速度极慢、广告多，甚至可能窃取隐私。建议使用收费的“机场”服务，哪怕是几块钱一个月的，体验也远好于免费工具。

**Q: 为什么连上 VPN 后国内软件（微信、抖音）变慢了？**
A: 优秀的客户端（Clash/Shadowrocket）默认使用 **“规则模式” (Rule Mode)**。即：访问国内网站不走代理，访问国外网站走代理。请确保你没有开启“全局模式” (Global Mode)，否则所有流量都会绕路国外，导致国内应用变慢。
